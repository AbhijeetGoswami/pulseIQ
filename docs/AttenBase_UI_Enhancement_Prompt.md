# AttenBase UI Design Enhancement Prompt (2026 SaaS Standard)

## Objective

Refine the existing AttenBase dashboard UI to achieve a premium 2026 B2B
SaaS aesthetic while preserving the current layout, spacing, and
information architecture. The goal is evolution, not redesign.

------------------------------------------------------------------------

# 1. Hero Banner Improvements

Replace the current flat mint/lime gradients with a branded visual
system.

### Create reusable SVG overlay assets featuring:

-   flowing contour/topographic lines
-   neural network nodes and connections
-   radar/signal rings
-   constellation dots
-   intelligence graph motifs
-   abstract signal waves

Requirements: - White artwork - Transparent background - 3--8% opacity -
Vector SVG - Responsive - Suitable for mint and lime gradients

Use layered mesh/radial gradients underneath instead of a single linear
gradient.

------------------------------------------------------------------------

# 2. KPI Cards

Keep white cards but modernize them.

Enhancements: - soft radial glow behind icon - slightly softer borders -
stronger typography hierarchy - subtle hover elevation - maintain
current spacing

Avoid colored backgrounds.

------------------------------------------------------------------------

# 3. Card Borders & Shadows

Replace visible borders with:

-   rgba(15,23,42,0.05)

Increase reliance on:

-   soft layered shadows
-   depth instead of outlines

------------------------------------------------------------------------

# 4. Pipeline Visualization

Transform the pipeline into a living system.

Add: - animated flowing dots between stages - status pulse animation -
active glow on running stage - subtle connector animations

Animation speed should remain slow and professional.

------------------------------------------------------------------------

# 5. Hero Card Branding

Create a unique AttenBase visual identity.

Signature design language: - mint gradients - white intelligence
patterns - subtle noise texture - mesh lighting - signal-inspired
overlays

This design language should be reused across: - Dashboard - Pipeline -
Future Analytics pages

------------------------------------------------------------------------

# 6. Sidebar

Reduce visual weight.

Recommendations: - slightly lighter background - softer separators -
maintain active gradient highlight - increase icon consistency

------------------------------------------------------------------------

# 7. Layout Rhythm

Avoid every section feeling identical.

Introduce rhythm by: - varying content density - alternating wider and
narrower sections - using visual breaks - preserving grid alignment

------------------------------------------------------------------------

# 8. Motion Design

Add tasteful micro-interactions.

Examples: - button hover lift - card elevation - shimmer on live
indicators - animated pipeline flow - smooth number transitions -
progress bar animation

Avoid excessive movement.

------------------------------------------------------------------------

# 9. Typography

Strengthen hierarchy.

Recommendations: - slightly larger section headings - stronger weight
for primary numbers - softer secondary text - maintain generous
whitespace

------------------------------------------------------------------------

# 10. Overall Design Principles

Target inspiration: - Linear - Stripe Dashboard - Vercel - Datadog -
Clerk - Retool - Grafana Cloud

The interface should communicate: - Intelligence - Reliability - Live
operations - Enterprise quality - Calm professionalism

Avoid: - heavy gradients - glassmorphism - excessive neon - clutter -
unnecessary decoration

The finished product should feel like a premium enterprise analytics
platform with a distinctive AttenBase brand identity rather than a
generic admin dashboard.
